# Changelog

Each changelog entry must include the version #, the PR #, the impact area, and a description of changes.

The version # is located in the constant `RUFAS_VERSION` in the file `RUFAS/task_manager.py`.

*Example Entry:*
- **0.0.0**: [#123](https://github.com/your-repo/pull/123) - Impact Area: Short description of the change or feature.

- **0.9.0**: [#1268](https://github.com/RuminantFarmSystems/MASM/pull/1968) - Changelog: Move the changelog from a Google sheet into a markdown document in the repository.
