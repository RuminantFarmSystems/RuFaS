# Changelog

Each changelog entry must include:
- Version #. This is located in the constant `RUFAS_VERSION` in the file `RUFAS/task_manager.py`.
- PR #. Just the number, omit "PR" and "#". Include a link to the Pull Request using the format "\[<PR #>\]\(\<link to PR\>\)"
- Impact area: What parts of the codebase, inputs, or outputs are affected by this PR? Some options are Animal Module, Manure Module, Output structure, whole code base, etc.
- A brief and concise description of change. Keep it short, yet informative. A few sentences should be enough. Avoid using broad and general statements such as "update xyz", it should be "update xyz to abc".


*Example Entry:*
- **0.0.0**: [123](https://github.com/your-repo/pull/123) - Impact Area: Short description of the change or feature.
