class RUFAS.routines.field.soil.carbon_cycling.residue_partition.ResiduePartition(soil_data: SoilData | None, field_size: float | None = None)#

Bases: object

Manages the partitioning of plant residues within the soil, affecting both plant-available nutrients and soil carbon storage.

Parameters#

soil_dataSoilData, optional

The SoilData object used by this module to track residue in the soil profile, creates new one if one is not provided.

field_sizefloat, optional

Used to initialize a SoilData object for this module to work with, if a pre-configured SoilData object is not provided (ha)

Attributes#

dataSoilData

The SoilData object used by this module to track residue in the soil profile.
